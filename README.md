

# 🦁 Trabalho 1: O Leão de Neméia — Encurtador de Links

Este projeto é o ponto de partida de uma jornada de aprendizado inspirada nos **12 Trabalhos de Hércules**, um desafio de programação proposto por um amigo.

Assim como o Leão de Neméia possuía uma pele impenetrável, o objetivo deste primeiro trabalho foi construir um encurtador de URLs com uma estrutura de dados robusta, persistente com SQLite e imune a colisões de códigos.

O repositório faz parte do hub central de desafios: [heracles-labours](https://github.com/matheusdcunha/heracles-labours).
[Nemean Lion](https://github.com/matheusdcunha/heracles-labours/tree/main/nemean-lion)

## 🚀 Como Usar a Aplicação

Siga os passos abaixo para rodar o encurtador de links localmente na sua máquina:

### 1. Clonar o Repositório

Abra o terminal e clone este projeto:

```bash
git clone https://github.com/ruanlov/Curtinho.git
cd Curtinho

```

### 2. Instalar as Dependências

O projeto utiliza o framework Flask. Certifique-se de ter o Python instalado e instale o pacote necessário:

```bash
pip install flask

```

### 3. Rodar o Servidor

Execute o arquivo principal do projeto para iniciar o servidor web:

```bash
python curtinho.py

```

*(Substitua `curtinho.py` pelo nome exato do seu arquivo Python, caso seja diferente).*

### 4. Acessar no Navegador

Com o servidor rodando, abra o seu navegador de internet e acesse o endereço local:

```text
http://localhost:5000/

```

Pronto! Agora basta colar qualquer URL longa no campo de texto para gerar o seu link encurtador persistente.
